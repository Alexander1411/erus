# erus

MSK Pain Clinic - Django Web Application

## Description

A Django-based web application for managing MSK Pain Clinic operations, including patient contact forms, appointment management, and clinic information.

## Features

- Patient contact form with consent management
- Privacy policy and terms of service
- Admin interface for managing contacts
- Responsive web design
